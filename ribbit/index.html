Twitter Clone

Uitleg opdracht

Een gebruiker kan een status update schrijven en plaatsen. Andere gebruikers kunnen daarop reageren en deze liken. Realiseer met jQuery of Javascript de volgende functionaliteit. Maak je geen zorgen dat de gebruiker, jij in dit geval, 1000x een thumbs up kan geven.

  • (5 punten) De gebruiker kan een status update posten en deze wordt direct op de timeline geplaatst (nieuwste boven)
  • (5 punten) Als een status update leeg is of meer is dan 140 characters, geef dan een foutmelding door middel van een alert
  • (5 punten) De gebruiker kan een status update waarderen door middel van thumbs up (+1) en thumbs down (-1).
  • (5 punten) De gebruiker kan een status update verwijderen.
